1. 이 웹페이지는 Cloud9에서 Ruby on rails 로 구성 후 구동했습니다.
2. 와이파이 모듈은 ESP8266입니다. 현재 고장난 것으로 추측되니 바로 보낸다고 열리지는 않을 겁니다. 새로 사거나, 펌웨어를 다시 밀어서 리셋(http://deneb21.tistory.com/269) 시켜야 하는 것으로 추측 됩니다.
3.  cloud9구동시 https:// 가 디폴트 값인데, 이걸 http://로 바꿔야 신호가 갔습니다.
4. 신호를 발신하는 컴퓨터와 신호를 받는 와이파이의 포트를 여셔야 할 겁니다. 현재 올린 것의 디폴트 값은 80번 포트 입니다.
5. RealWifiMotor가 아두이노에 들어갈 실제 코드 입니다.
6. 그외에 led는 와이파이 실험용 아두이노코드, simplemo는 모터 구동 코드 입니다. realwifimotor는 이 둘을 조합한 것입니다.
7. led 코드의 출처는 http://deneb21.tistory.com/274 입니다.
8. app- view - home -index.html.erb 의 238번대 코드, 
    $.get("http://192.168.2.17:80/", {pin:p}); // execute get request
에서 안의 주소값과 포트 값을 실제 와이파이 주소 값으로 바꿔야 합니다. 와이파이 모듈이 정상일 시, 아두이노 구동 시 시리얼 모니터에 나타나는 와이파이 주소가
그것입니다.
